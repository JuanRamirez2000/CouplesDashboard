# An app meant for couples to connect

## Description

The app is a dashboard for couples to connect ideas together for stuff like dates, gifts, future plans, etc. The main features for this is realtime updates, user accounts, and messaging to coordinate their plans together


The app is made around a backend as a service known as convex.

## The Stack

This project currently works under NextJS. For specifics

### Frontend tech

- ReactJS + TypescriptJS
- TailwindCSS

### Backend tech

- Convex
- Clerk (auth)

### Infra

- Vercel
